B2B Prospecting with Ad Intelligence Data

Published: January 22, 2026 • Reading time: 6 min

LinkedIn Ad Library activity reveals companies that are ready to buy. Companies running active ad campaigns have budget, intent, and decision-making structures in place. In this guide, I'll show you how to find these warm prospects and turn them into clients without technical skills.

Ad Activity as Intent Signal

Unlike traditional firmographic data, ad activity indicates active business growth initiatives:

  • Budget availability: Companies running ads have allocated marketing budgets
  • Growth mindset: Active advertising suggests expansion plans
  • Decision-making structure: Ad campaigns require approval processes
  • Measurable goals: Ad spend implies performance expectations

Technical Implementation: Data Enrichment Pipeline

Our prospecting pipeline enriches traditional B2B data with ad intelligence:

class ProspectingEnricher:
    def __init__(self, ad_data_source, contact_enrichment_service):
        self.ad_data = ad_data_source
        self.contact_service = contact_enrichment_service
    
    def enrich_prospect(self, company_info):
        """
        Enrich prospect with ad intelligence data.
        
        Args:
            company_info: Basic company information
        
        Returns:
            Enriched prospect profile with ad data
        """
        # Step 1: Check for active LinkedIn ads
        ad_data = self.ad_data.get_company_ads(company_info['name'])
        
        # Step 2: Calculate prospect intent score
        intent_score = self.calculate_intent_score(ad_data)
        
        # Step 3: Get decision maker contacts
        contacts = self.contact_service.get_decision_makers(
            company_info['name'], 
            company_info.get('industry')
        )
        
        # Step 4: Combine all data
        enriched_prospect = {
            'company': company_info,
            'ad_activity': ad_data,
            'intent_score': intent_score,
            'contacts': contacts,
            'prospect_ranking': self.rank_prospect(intent_score, contacts)
        }
        
        return enriched_prospect
    
    def calculate_intent_score(self, ad_data):
        """
        Calculate prospect intent based on ad activity.
        
        Factors considered:
        - Ad volume (more ads = higher intent)
        - Ad diversity (multiple formats = serious investment)
        - Ad duration (long-running ads = sustained interest)
        """
        if not ad_data or ad_data['Has_Ads'] == 'No':
            return 0
        
        ad_count = int(ad_data['Ad_Count'])
        ad_formats = ad_data['Ad_Formats']
        
        # Base score on ad volume
        if ad_count >= 50:
            volume_score = 5
        elif ad_count >= 20:
            volume_score = 4
        elif ad_count >= 10:
            volume_score = 3
        elif ad_count >= 5:
            volume_score = 2
        else:
            volume_score = 1
        
        # Bonus for format diversity
        formats = [f.strip() for f in ad_formats.split(',') if f.strip()]
        format_bonus = min(len(formats), 2)  # Max 2 bonus points
        
        return volume_score + format_bonus
    
    def rank_prospect(self, intent_score, contacts):
        """
        Rank prospect based on intent and contact quality.
        """
        if not contacts:
            return "Low"  # No contacts = low rank regardless of intent
        
        contact_quality = self.assess_contact_quality(contacts)
        
        if intent_score >= 6 and contact_quality == "High":
            return "Priority"
        elif intent_score >= 4 and contact_quality in ["High", "Medium"]:
            return "High"
        elif intent_score >= 2:
            return "Medium"
        else:
            return "Low"

Prospect Segmentation Strategy

We segment prospects based on ad activity and contact quality:

def segment_prospects(prospects):
    """
    Segment prospects based on ad activity and contact quality.
    """
    segments = {
        'priority': [],  # High intent + decision maker contacts
        'warm': [],      # Medium-high intent + good contacts
        'research': [],  # High intent + limited contacts
        'monitor': []    # Low activity but potential
    }
    
    for prospect in prospects:
        intent = prospect['intent_score']
        contact_quality = assess_contact_quality(prospect['contacts'])
        
        if intent >= 6 and contact_quality == 'High':
            segments['priority'].append(prospect)
        elif intent >= 4 and contact_quality in ['High', 'Medium']:
            segments['warm'].append(prospect)
        elif intent >= 3:
            segments['research'].append(prospect)
        else:
            segments['monitor'].append(prospect)
    
    return segments

def export_to_apollo_format(prospects):
    """
    Export prospects in Apollo-compatible format.
    """
    apollo_fields = [
        'Organization Name',
        'Website',
        'Industry',
        'Employee Count',
        'City',
        'State',
        'Country',
        'Person Name',
        'Person Email',
        'Person Title',
        'Person LinkedIn URL',
        'Ad Activity Score',
        'Intent Score',
        'Prospect Rank'
    ]
    
    rows = []
    for prospect in prospects:
        for contact in prospect['contacts']:
            row = {
                'Organization Name': prospect['company']['name'],
                'Website': prospect['company'].get('website', ''),
                'Industry': prospect['company'].get('industry', ''),
                'Employee Count': prospect['company'].get('employee_count', ''),
                'City': prospect['company'].get('city', ''),
                'State': prospect['company'].get('state', ''),
                'Country': prospect['company'].get('country', ''),
                'Person Name': contact['name'],
                'Person Email': contact['email'],
                'Person Title': contact['title'],
                'Person LinkedIn URL': contact['linkedin_url'],
                'Ad Activity Score': prospect['ad_activity']['Ad_Count'],
                'Intent Score': prospect['intent_score'],
                'Prospect Rank': prospect['prospect_ranking']
            }
            rows.append(row)
    
    return {'fields': apollo_fields, 'rows': rows}

Key Principles for Effective Outreach

Follow these key principles for effective outreach:

1. Research-Based Personalization

Rather than generic messaging, research each prospect's specific situation:

  • Recent company news or funding
  • Current ad campaigns and messaging
  • Industry challenges and trends
  • Specific role-based pain points

2. Outcome-Focused Messaging

Lead with specific outcomes rather than generic benefits:

  • "Increase demo-to-customer conversion by 32%" instead of "Improve conversions"
  • "Reduce cost per qualified lead by 25%" instead of "Lower costs"
  • "Track ad spend ROI by customer lifetime value" instead of "Measure ROI"

3. Contextual Relevance

Tie your message to the prospect's actual business situation:

  • Recent funding rounds
  • Competitive pressures
  • Market expansion plans
  • Current marketing challenges

4. Low-Commitment Next Steps

Make it easy for prospects to engage:

  • "Quick 5-minute question" instead of "30-minute demo"
  • "Share a quick benchmark" instead of "Schedule presentation"
  • "See if this applies to you" instead of "Sign up now"
