LinkedIn Ad Intelligence: Technical Guide for Agencies

Published: January 30, 2026 • Reading time: 8 min

LinkedIn's Ad Library is a goldmine for competitive intelligence, but extracting data at scale requires technical know-how. In this LinkedIn Ad Library guide, I'll walk through the technical implementation we use to gather ad intelligence from curated ad libraries for 500+ B2B companies monthly. Learn how to leverage the LinkedIn ads library for competitive analysis and business intelligence.

Understanding LinkedIn's Ad Library Structure

The Ad Library is publicly accessible without authentication, but LinkedIn employs several anti-scraping measures:

  • Dynamic content loaded via JavaScript
  • Rate limiting based on request patterns
  • Frequent DOM structure changes
  • Browser fingerprinting

The key endpoint for company searches is:

https://www.linkedin.com/ad-library/search?keyword={company_name}&countries=US
Important: Use countries=US instead of geographicAreaId for better results. This was a critical discovery that increased our success rate from 15% to 85%.

Technical Implementation with Playwright

We use Playwright for browser automation due to its ability to handle dynamic content and mimic human behavior:

async def get_ad_data(context, company_name, linkedin_url=""):
    result = {
        "Has_Ads": "No",
        "Ad_Count": "0",
        "Ad_Formats": "",
        "Sample_Ad_Url": "",
        "Ad_Library_Search_Url": ""
    }

    keyword_url = f"https://www.linkedin.com/ad-library/search?keyword={urllib.parse.quote(company_name)}"
    result["Ad_Library_Search_Url"] = keyword_url

    page = await context.new_page()
    try:
        await page.goto(keyword_url, timeout=30000)
        await asyncio.sleep(3)

        html = await page.content()

        # Get accurate ad count from "X ads match your search criteria" header
        count_match = re.search(r'(\d+)\s+ads?\s+match', html, re.IGNORECASE)
        if count_match:
            ad_count = int(count_match.group(1))
            result["Has_Ads"] = "Yes"
            result["Ad_Count"] = str(ad_count)

            # Scroll to load more ad formats
            for _ in range(3):
                await page.evaluate("window.scrollTo(0, document.body.scrollHeight)")
                await asyncio.sleep(0.5)

            html = await page.content()
            format_matches = re.findall(r'(Video|Carousel|Document|Image|Event) Ad', html)
            result["Ad_Formats"] = ", ".join(sorted(set(format_matches))) if format_matches else ""

            # Get sample ad URL
            sample_match = re.search(r'href="(/ad-library/detail/[^"]+)"', html)
            if sample_match:
                result["Sample_Ad_Url"] = "https://www.linkedin.com" + sample_match.group(1)

    except Exception as e:
        print(f"  Error: {e}")
    finally:
        await page.close()

    return result

Handling Rate Limits and Anti-Bot Measures

LinkedIn implements rate limiting through several mechanisms:

  1. Request frequency: More than 1 request per 2-3 seconds triggers temporary blocks
  2. Browser fingerprinting: Consistent headers and behaviors flag bots
  3. Behavioral analysis: Unnatural scrolling and clicking patterns

Our mitigation strategies:

  • Randomized delays between 2-5 seconds
  • Rotating user agents
  • Simulated human-like scrolling
  • Session persistence to appear as returning visitor

Data Processing Pipeline

After collecting raw ad data, we process it through several stages:

  1. Filtering: Remove companies with fewer than 10 active ads (indicates serious ad spend)
  2. Deduplication: Remove major corporations and unicorns
  3. Classification: Verify B2B status using industry tags
  4. Enrichment: Add decision-maker contact information

Performance Optimization

Processing 10,000+ companies efficiently requires optimization:

  • Batch processing: Process 10 companies before saving to disk
  • Error recovery: Continue processing if individual requests fail
  • Progress tracking: Log progress for debugging and monitoring

Our current pipeline processes 500 companies in approximately 4 hours with 85% success rate.

Legal and Ethical Considerations

All data collected comes from LinkedIn's public Ad Library, which is intended for transparency purposes. However, we recommend:

  • Reviewing LinkedIn's Terms of Service regularly
  • Implementing respectful scraping practices
  • Using data only for legitimate business purposes

Implementing Research-Based Outreach

To maximize the impact of ad intelligence data, agencies should implement research-based outreach methodology:

1. Research-Based Personalization

Instead of generic outreach, use the ad intelligence data to craft personalized messages:

# Example: Using ad data for personalization
def create_personalized_outreach(prospect):
    """
    Create personalized outreach using research-based methodology.
    """
    # Use actual ad activity to personalize
    ad_count = int(prospect['ad_activity']['Ad_Count'])
    ad_formats = prospect['ad_activity']['Ad_Formats']
    sample_ad_url = prospect['ad_activity']['Sample_Ad_Url']

    # Craft message based on specific observations
    if ad_count > 20:
        subject = f"Quick question about your {ad_count}-ad LinkedIn strategy"
        body = f"""
        Hi {prospect['contact']['first_name']},

        I noticed {prospect['company']['name']} is running {ad_count} active LinkedIn ads - impressive scale!

        Quick question: Are you tracking which ad formats drive the highest LTV customers vs. just leads?

        Most B2B companies optimize for lead volume but miss the connection to actual revenue impact.

        If that's relevant, I can share a simple framework we built for measuring ad ROI by customer value.

        Worth a 5-min chat?

        Best,
        [Your Name]
        """
    elif 'Video' in ad_formats:
        subject = f"Thoughts on your LinkedIn video ad performance?"
        body = f"""
        Hi {prospect['contact']['first_name']},

        Saw your video ads on LinkedIn for {prospect['company']['name']} - video is a great way to stand out.

        Question: Are you seeing good engagement-to-conversion rates on video vs. static ads?

        We've helped companies in your space improve video ad conversion by 28% by optimizing the follow-up sequence.

        Curious if this is something you're focused on right now?

        Best,
        [Your Name]
        """

    return {'subject': subject, 'body': body}

2. Outcome-Focused Messaging

Focus on specific outcomes rather than generic benefits:

  • "Increase demo-to-customer conversion by 32%" instead of "Improve conversions"
  • "Reduce cost per qualified lead by 25%" instead of "Lower costs"
  • "Track ad spend ROI by customer lifetime value" instead of "Measure ROI"

3. Contextual Relevance

Tie your message to the prospect's actual business situation:

  • Recent funding rounds
  • Competitive pressures
  • Market expansion plans
  • Current marketing challenges

4. Low-Commitment Next Steps

Make it easy for prospects to engage:

  • "Quick 5-minute question" instead of "30-minute demo"
  • "Share a quick benchmark" instead of "Schedule presentation"
  • "See if this applies to you" instead of "Sign up now"

The technical implementation of ad intelligence gathering is complex, but when combined with research-based outreach methodology, agencies can achieve significantly better results. By focusing on companies with active ad spend and personalizing outreach based on their actual activities, agencies can dramatically improve their response rates and conversion.
